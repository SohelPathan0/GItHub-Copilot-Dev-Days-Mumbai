## 📌 About the Event

**GitHub Copilot Dev Days Mumbai** was a hands-on, in-person, community-led event focused on helping developers explore real-world AI-assisted coding using **GitHub Copilot**. Part of the global **GitHub Copilot Dev Days** series, this event brought together 83 developers from across Mumbai to learn, build, and connect.

The event covered practical, real-world usage of GitHub Copilot across multiple IDEs, workflows, and languages — giving every attendee skills they could apply immediately after the event.

## 🎓 Certificate Verification

Every participant received a **digital certificate** with a unique verification link and QR code, both pointing to the GitHub Pages verification system hosted in this repository.

### 🌐 Verification Page

> **https://SohelPathan0.github.io/certificate-verification/**

### 🔗 Verify a Specific Certificate

```
https://SohelPathan0.github.io/certificate-verification/?id=GCDM-081
```

Simply replace `GCDM-081` with the certificate ID printed on your certificate.

### How It Works

```
Certificate (PDF / Image)
  ├── Unique URL  →  ?id=GCDM-XXX  ──┐
  └── QR Code     →  same URL      ──┤
                                      ↓
                        Page loads & reads ?id= from URL
                                      ↓
                        Fetches data.json from this repo
                                      ↓
                        Looks up certificate ID
                                      ↓
                 ✅ Shows name, event, role, status
                           OR
                 ❌ Certificate not found
```

### Certificate Record Format (`data.json`)

```json
{
  "certificates": [
    {
      "id": "GCDM-001",
      "name": "Participant Full Name",
      "event": "GitHub Copilot Dev Days | Mumbai",
      "role": "Participant"
    }
  ]
}
```
